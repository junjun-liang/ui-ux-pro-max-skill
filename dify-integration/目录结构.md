## 📝 关键文件说明

### dify-integration/ui_ux_pro_max.py

**作用**: Dify工具的Python实现

**核心类**:
- `BM25`: BM25搜索引擎
- `DesignSystemGenerator`: 设计系统生成器

**核心函数**:
- `generate_design_system()`: 生成设计系统
- `search_domain()`: 搜索特定域
- `search_stack_guidelines()`: 搜索技术栈指南

**配置**:
```python
DATA_DIR = Path(__file__).parent / "data"  # 指向data/目录
CSV_CONFIG = {...}  # CSV文件配置
STACK_CONFIG = {...}  # 技术栈配置
```

### dify-integration/provider.yaml

**作用**: 工具提供者配置

**配置项**:
```yaml
identity:
  name: "ui_ux_pro_max"
```

### dify-integration/tools.yaml

**作用**: 工具定义和参数配置

**工具列表**:
1. `generate_design_system`: 生成设计系统
2. `search_domain`: 搜索特定域
3. `search_stack_guidelines`: 搜索技术栈指南

**参数定义**:

## 🚀 部署流程

### Docker部署流程

```
1. 启动Dify
   cd dify/docker
   docker compose up -d

2. 进入API容器
   docker exec -it dify-api-1 bash

3. 创建工具目录
   cd /app/api/core/tools/provider/builtin
   mkdir -p ui_ux_pro_max

4. 复制工具文件
   docker cp ui_ux_pro_max.py dify-api-1:/app/api/core/tools/provider/builtin/ui_ux_pro_max/
   docker cp provider.yaml dify-api-1:/app/api/core/tools/provider/builtin/ui_ux_pro_max/
   docker cp tools.yaml dify-api-1:/app/api/core/tools/provider/builtin/ui_ux_pro_max/

5. 复制数据文件
   docker cp src/ui-ux-pro-max/data/ dify-api-1:/app/api/core/tools/provider/builtin/ui_ux_pro_max/data/

6. 重启API服务
   docker restart dify-api-1

7. 在Dify中添加工具
   - 打开Dify管理后台
   - 创建/编辑应用
   - 添加"UI/UX Pro Max"工具
```

### 本地开发部署流程

```
1. 克隆Dify仓库
   git clone https://github.com/langgenius/dify.git
   cd dify

2. 安装依赖
   cd api
   pip install -r requirements.txt

3. 创建工具目录
   cd core/tools/provider/builtin
   mkdir -p ui_ux_pro_max

4. 复制工具文件
   cp /path/to/ui-ux-pro-max-skill/dify-integration/ui_ux_pro_max.py .
   cp /path/to/ui-ux-pro-max-skill/dify-integration/provider.yaml .
   cp /path/to/ui-ux-pro-max-skill/dify-integration/tools.yaml .

5. 复制数据文件
   mkdir -p data
   cp /path/to/ui-ux-pro-max-skill/src/ui-ux-pro-max/data/*.csv data/
   cp -r /path/to/ui-ux-pro-max-skill/src/ui-ux-pro-max/data/stacks data/

6. 修改数据路径
   # 编辑ui_ux_pro_max.py
   DATA_DIR = Path(__file__).parent / "data"

7. 启动API服务
   cd /path/to/dify/api
   python main.py
```

## 🔧 配置修改

### 修改数据路径

如果需要修改CSV数据文件的位置：

```python
# 在ui_ux_pro_max.py中修改
# 原始代码
DATA_DIR = Path(__file__).parent.parent / "src" / "ui-ux-pro-max" / "data"

# 修改为
DATA_DIR = Path(__file__).parent / "data"  # Docker部署
# 或
DATA_DIR = Path("/app/data/ui-ux-pro-max")  # 自定义路径
```

### 修改搜索参数

如果需要调整搜索行为：

```python
# 在ui_ux_pro_max.py中修改
MAX_RESULTS = 3  # 默认最大结果数

# BM25参数调整
class BM25:

        self.k1 = k1
        self.b = b
```
