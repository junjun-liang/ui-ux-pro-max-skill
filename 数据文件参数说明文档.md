## 概述

UI/UX Pro Max的数据目录包含16个CSV文件，涵盖161种产品类型、67种UI风格、161种配色方案、57种字体配对、25种图表类型、99条UX指南等设计资源。

### 文件分类

| 分类 | 文件数量 | 说明 |
|------|----------|------|
| **核心数据** | 4个 | 产品类型、UI风格、配色方案、字体配对 |
| **设计系统** | 3个 | 推理规则、图表类型、落地页模式 |
| **指南** | 4个 | UX指南、Google字体、图标库、应用界面 |
| **性能** | 1个 | React性能指南 |
| **技术栈** | 1个 | React Native指南（可扩展） |

## 参数使用技巧

### 1. 搜索关键词优化

**技巧**: 使用多个相关关键词提高搜索准确性

**示例**:
```python
# 好的搜索
search("SaaS modern professional dashboard", "style", 3)

# 不好的搜索
search("SaaS", "style", 3)
```

### 2. 域自动检测

**技巧**: 让系统自动检测最相关的域

**示例**:
```python
# 自动检测域
search("glassmorphism", None, 3)
# 系统会自动检测到 "style" 域

# 指定域
search("glassmorphism", "style", 3)
```

### 3. 多域并行搜索

**技巧**: 同时搜索多个域获取全面信息

**示例**:
```python
# 并行搜索多个域
results = {
    "style": search("SaaS", "style", 3),
    "color": search("SaaS", "color", 2),
    "typography": search("SaaS", "typography", 2),
    "landing": search("SaaS", "landing", 2)
}
```

### 4. 推理规则应用

**技巧**: 使用推理规则优化设计决策

**示例**:
```python
# 获取推理规则
rule = find_reasoning_rule("Beauty/Spa Service")

# 应用风格优先级
style_priority = rule["Style_Priority"].split("+")
# ["Soft UI Evolution", "Neumorphism"]

# 应用风格优先级到搜索
search("beauty spa", "style", 3, style_priority)
```

### 5. 检查清单验证

**技巧**: 使用检查清单确保实现质量

**示例**:
```python
# 获取风格检查清单
style = search("Minimalism", "style", 1)
checklist = style["results"][0]["Implementation Checklist"]

# 验证实现
for item in checklist.split(";"):
    if not check_implemented(item):
        print(f"Missing: {item}")
```
