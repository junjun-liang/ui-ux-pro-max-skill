# UI/UX Pro Max - 软件设计架构文档

## 项目概述

UI/UX Pro Max 是一个AI驱动的UI/UX设计智能工具包，为跨多个平台和框架构建专业UI/UX提供设计智能。该项目通过可搜索的数据库、推理引擎和模板系统，为AI编码助手提供设计指导。

### 核心特性

- **161个行业特定的推理规则** - 针对不同产品类型的设计决策
- **67种UI风格** - Glassmorphism、Claymorphism、Minimalism、Brutalism等
- **161种配色方案** - 与产品类型1:1对齐的行业特定配色
- **57种字体配对** - 精心策划的排版组合
- **25种图表类型** - 仪表盘和分析可视化推荐
- **13种技术栈支持** - React、Next.js、Vue、Svelte、SwiftUI、React Native、Flutter等
- **99条UX指南** - 最佳实践、反模式和无障碍规则
- **智能设计系统生成器** - v2.0旗舰功能

## 核心组件详解

### 1. BM25搜索引擎 (core.py)

**功能**: 实现BM25排名算法用于文本搜索

**核心类**:
```python
class BM25:
    """BM25排名算法用于文本搜索"""
    
    def __init__(self, k1=1.5, b=0.75):
        self.k1 = k1  # 词频饱和参数
        self.b = b    # 文档长度归一化参数
        self.corpus = []
        self.doc_lengths = []
        self.avgdl = 0
        self.idf = {}
        self.doc_freqs = defaultdict(int)
        self.N = 0
```

**工作流程**:
1. **分词** (tokenize): 小写化、去除标点、过滤短词
2. **索引构建** (fit): 构建文档语料库、计算文档长度、统计词频
3. **评分** (score): 使用BM25公式计算查询与文档的相关性

**BM25公式**:
```
score(D,Q) = Σ IDF(qi) * (f(qi,D) * (k1 + 1)) / (f(qi,D) + k1 * (1 - b + b * |D|/avgdl))
```

### 2. 域检测引擎 (detect_domain)

**功能**: 自动检测查询最相关的域

**支持的域**:
| 域 | 用途 | 关键词示例 |
|----|------|-----------|
| `color` | 配色方案 | color, palette, hex, rgb, token, semantic |
| `chart` | 图表类型 | chart, graph, visualization, trend, bar, pie |
| `landing` | 落地页模式 | landing, page, cta, conversion, hero, testimonial |
| `product` | 产品类型推荐 | saas, ecommerce, fintech, healthcare, gaming |
| `style` | UI风格 | style, design, ui, minimalism, glassmorphism |
| `ux` | UX最佳实践 | ux, usability, accessibility, wcag, touch |
| `typography` | 字体配对 | font pairing, typography, heading font, body font |
| `google-fonts` | Google字体 | google font, font family, variable font |
| `icons` | 图标库 | icon, icons, lucide, heroicons, svg icon |
| `react` | React性能 | react, next.js, suspense, memo, rerender |
| `web` | 应用界面指南 | aria, focus, outline, semantic, virtualize |

**算法**: 基于关键词匹配的评分系统

```python
def detect_domain(query):
    query_lower = query.lower()
    domain_keywords = {
        "color": ["color", "palette", "hex", "#", "rgb", ...],
        "chart": ["chart", "graph", "visualization", ...],
        # ... 其他域
    }
    scores = {domain: sum(1 for kw in keywords if kw in query_lower) 
               for domain, keywords in domain_keywords.items()}
    return max(scores, key=scores.get)
```

```python
class DesignSystemGenerator:
    def __init__(self):
        self.reasoning_data = self._load_reasoning()
    
    def _multi_domain_search(self, query: str, style_priority: list = None) -> dict:
        """执行多域搜索"""
        results = {}
        for domain, config in SEARCH_CONFIG.items():
            if domain == "style" and style_priority:
                priority_query = " ".join(style_priority[:2])
                combined_query = f"{query} {priority_query}"
                results[domain] = search(combined_query, domain, config["max_results"])
            else:
                results[domain] = search(query, domain, config["max_results"])
        return results
    
    def _apply_reasoning(self, category: str, search_results: dict) -> dict:
        """应用推理规则"""
        rule = self._find_reasoning_rule(category)
        decision_rules = json.loads(rule.get("Decision_Rules", "{}"))
        return {
            "pattern": rule.get("Recommended_Pattern", ""),
            "style_priority": [s.strip() for s in rule.get("Style_Priority", "").split("+")],
            "color_mood": rule.get("Color_Mood", ""),
            "typography_mood": rule.get("Typography_Mood", ""),
            "key_effects": rule.get("Key_Effects", ""),
            "anti_patterns": rule.get("Anti_Patterns", ""),
            "decision_rules": decision_rules
        }
    
    def generate(self, query: str, project_name: str = None) -> dict:
        """生成完整设计系统推荐"""
        # 步骤1: 搜索产品类型
        product_result = search(query, "product", 1)
        category = product_result.get("results", [{}])[0].get("Product Type", "General")
        
        # 步骤2: 获取推理规则
        reasoning = self._apply_reasoning(category, {})
        
        # 步骤3: 多域搜索
        search_results = self._multi_domain_search(query, reasoning.get("style_priority", []))
        
        # 步骤4: 选择最佳匹配
        best_style = self._select_best_match(search_results["style"], reasoning.get("style_priority", []))
        best_color = search_results["color"][0]
        best_typography = search_results["typography"][0]
        best_landing = search_results["landing"][0]
        
        # 步骤5: 构建最终推荐
        return {
            "project_name": project_name or query.upper(),
            "category": category,
            "pattern": {...},
            "style": {...},
            "colors": {...},
            "typography": {...},
            "key_effects": combined_effects,
            "anti_patterns": reasoning.get("anti_patterns", ""),
            "decision_rules": reasoning.get("decision_rules", {})
        }
```

### BM25搜索算法详解

**算法原理**:
BM25是一种基于概率的检索函数，用于评估文档与查询的相关性。它是对TF-IDF的改进，引入了文档长度归一化和词频饱和参数。

**公式**:
```
score(D,Q) = Σ IDF(qi) * (f(qi,D) * (k1 + 1)) / (f(qi,D) + k1 * (1 - b + b * |D|/avgdl))
```

其中:
- `D`: 文档
- `Q`: 查询
- `qi`: 查询中的第i个词
- `f(qi,D)`: 词qi在文档D中的词频
- `|D|`: 文档D的长度
- `avgdl`: 平均文档长度
- `k1`: 词频饱和参数 (通常1.2-2.0)
- `b`: 长度归一化参数 (通常0.75)
- `IDF(qi)`: 逆文档频率

**IDF计算**:
```
IDF(qi) = log((N - df(qi) + 0.5) / (df(qi) + 0.5) + 1)
```

其中:
- `N`: 文档总数
- `df(qi)`: 包含词qi的文档数

**实现代码**:
```python
class BM25:
    def __init__(self, k1=1.5, b=0.75):
        self.k1 = k1
        self.b = b
        self.corpus = []
        self.doc_lengths = []
        self.avgdl = 0
        self.idf = {}
        self.doc_freqs = defaultdict(int)
        self.N = 0
    
    def tokenize(self, text):
        """分词: 小写化、去除标点、过滤短词"""
        text = re.sub(r'[^\w\s]', ' ', str(text).lower())
        return [w for w in text.split() if len(w) > 2]
    
    def fit(self, documents):
        """构建索引"""
        self.corpus = [self.tokenize(doc) for doc in documents]
        self.N = len(self.corpus)
        self.doc_lengths = [len(doc) for doc in self.corpus]
        self.avgdl = sum(self.doc_lengths) / self.N
        
        # 统计文档频率
        for doc in self.corpus:
            seen = set()
            for word in doc:
                if word not in seen:
                    self.doc_freqs[word] += 1
                    seen.add(word)
        
        # 计算IDF
        for word, freq in self.doc_freqs.items():
            self.idf[word] = log((self.N - freq + 0.5) / (freq + 0.5) + 1)
    
    def score(self, query):
        """评分"""
        query_tokens = self.tokenize(query)
        scores = []
        
        for idx, doc in enumerate(self.corpus):
            score = 0
            doc_len = self.doc_lengths[idx]
            term_freqs = defaultdict(int)
            for word in doc:
                term_freqs[word] += 1
            
            for token in query_tokens:
                if token in self.idf:
                    tf = term_freqs[token]
                    idf = self.idf[token]
                    numerator = tf * (self.k1 + 1)
                    denominator = tf + self.k1 * (1 - self.b + self.b * doc_len / self.avgdl)
                    score += idf * numerator / denominator
            
            scores.append((idx, score))
        
        return sorted(scores, key=lambda x: x[1], reverse=True)
```

### 域检测算法

**原理**: 基于关键词匹配的评分系统

**实现**:
```python
def detect_domain(query):
    query_lower = query.lower()
    
    domain_keywords = {
        "color": ["color", "palette", "hex", "#", "rgb", "token", "semantic"],
        "chart": ["chart", "graph", "visualization", "trend", "bar", "pie"],
        "landing": ["landing", "page", "cta", "conversion", "hero", "testimonial"],
        "product": ["saas", "ecommerce", "fintech", "healthcare", "gaming"],
        "style": ["style", "design", "ui", "minimalism", "glassmorphism"],
        "ux": ["ux", "usability", "accessibility", "wcag", "touch"],
        "typography": ["font pairing", "typography", "heading font", "body font"],
        "google-fonts": ["google font", "font family", "variable font"],
        "icons": ["icon", "icons", "lucide", "heroicons", "svg icon"],
        "react": ["react", "next.js", "suspense", "memo", "rerender"],
        "web": ["aria", "focus", "outline", "semantic", "virtualize"]
    }
    
    # 计算每个域的匹配分数
    scores = {domain: sum(1 for kw in keywords if kw in query_lower) 
               for domain, keywords in domain_keywords.items()}
    
    # 返回分数最高的域
    best = max(scores, key=scores.get)
    return best if scores[best] > 0 else "style"
```

## 扩展性

### 添加新的UI风格

1. 在`styles.csv`中添加新行
2. 填充所有必需列
3. 运行同步脚本:
```bash
python3 src/ui-ux-pro-max/data/_sync_all.py
```

### 添加新的产品类型

1. 在`products.csv`中添加新行
2. 在`ui-reasoning.csv`中添加对应推理规则
3. 在`colors.csv`中添加对应配色方案
4. 运行同步脚本

### 添加新的技术栈

1. 在`src/ui-ux-pro-max/data/stacks/`中创建新CSV文件
2. 在`core.py`中添加配置:
```python
STACK_CONFIG = {
    "react-native": {"file": "stacks/react-native.csv"},
    "new-stack": {"file": "stacks/new-stack.csv"},  # 新增
}
```
3. 更新`AVAILABLE_STACKS`列表

### 添加新的AI平台支持

1. 在`src/ui-ux-pro-max/templates/platforms/`中创建新JSON文件:
```json
{
  "title": "New Platform",
  "description": "Description of the platform",
  "commands": {
    "skill": "/new-platform",
    "workflow": "/ui-ux-pro-max"
  }
}
```

2. 在`cli/src/types/index.ts`中添加新平台:
```typescript
export const AI_TYPES = [
  'claude',
  'cursor',
  'new-platform',  // 新增
  // ...
] as const;
```

3. 在`cli/src/utils/detect.ts`中添加检测逻辑

## 故障排除

### 常见问题

#### 1. Python未安装
**错误**: `python3: command not found`

**解决**:
```bash
# macOS
brew install python3

# Ubuntu/Debian
sudo apt update && sudo apt install python3

# Windows
winget install Python.Python.3.12
```

#### 2. CLI安装失败
**错误**: `npm ERR! code EACCES`

**解决**:
```bash
# 使用sudo
sudo npm install -g uipro-cli

# 或使用npx
npx uipro-cli init --ai claude
```

#### 3. 搜索无结果
**错误**: `Found: 0 results`

**解决**:
- 检查查询关键词
- 尝试不同的域: `--domain style`
- 使用更通用的查询词

#### 4. 设计系统生成失败
**错误**: `File not found: ui-reasoning.csv`

**解决**:
```bash
# 运行同步脚本
cd src/ui-ux-pro-max/data
python3 _sync_all.py
```
